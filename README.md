## Basic calculator on Golang

How to start:
```zsh
$ go mod tidy
$ go run cmd/main.go
```
How to build:
```zsh
$ go build -v ./cmd/main.go
$ ./main
```

How to start with docker:
```zsh
$ docker build -t go-calculator-app .
$ docker run -it go-calculator-app
```
